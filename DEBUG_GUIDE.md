# 🔍 播放器调试指南

## ✅ 已改进的功能

1. **拖放增强** - 添加了拖放时的视觉反馈
2. **详细日志** - 浏览器控制台有完整的加载过程
3. **错误显示** - 如果视频加载失败会显示错误信息
4. **文件名显示** - 显示正在播放的文件名
5. **更好的清理** - 确保事件监听器正确清理

## 🚀 测试步骤

### 1. 启动应用
```bash
npm run dev
```

### 2. 打开浏览器控制台
- 按 **F12** 打开开发者工具
- 点击 **Console** 标签页

### 3. 测试打开文件

#### 方式 A - 点击图标
1. 点击底部 📂 图标
2. 选择一个 MP4 文件（H.264 编码）
3. 观察控制台输出

#### 方式 B - 拖放文件
1. 从资源管理器拖一个视频文件
2. 拖到播放器窗口（应该看到蓝色背景
3. 松开鼠标
4. 观察控制台输出

## 📋 应该看到的控制台日志

### 拖放时：

```
✓ Video playing
✓ Video paused
```

## 🎯 检查清单

- [ ] 拖放时有蓝色背景
- [ ] 控制台显示 "Drop event triggered"
- [ ] 控制台显示 "Loading file: ..."
- [ ] 视频显示在屏幕上
- [ ] 播放按钮点击有反应
- [ ] 进度条更新
- [ ] 文件名显示在左上角

## 💡 提示

1. **使用 MP4 (H.264)** - 浏览器最支持的格式
2. **检查文件大小** - 先用小文件测试
3. **查看控制台** - 所有信息都在那里
4. **手动播放** - 如果自动播放被阻止，手动点击播放按钮
