# ai-mplayer - Windows 11 智能播放器

一个功能强大的Windows 11智能媒体播放器，支持实时解码播放、AI增强、字幕处理等功能。

## 技术栈

- **Tauri**: 轻量级原生应用框架
- **React**: UI框架
- **libmpv**: 核心媒体播放引擎（实时解码）
- **Tailwind CSS**: 样式框架

## 核心功能

1. 实时解码播放，不转码
2. 自动检测和解码器管理
3. 硬件检测和自动配置
4. AI增强处理
5. 多字幕支持（内置/外挂/ASR）
6. AI字幕识别和翻译
7. P2P弹幕
8. Ollama/大模型API集成
9. 国际化多语言
10. 国内源优先
