# 🪟 Tauri 窗口模式（方案 1）

## 已完成的配置

✅ Cargo.toml → 使用 web 版本配置
✅ main.rs → 使用 web 版本后端
✅ libmpv 文件 → 已就位（暂时不用）

## 🚀 现在运行 Tauri 窗口模式

```bash
npm run tauri:dev
```

这将打开一个原生的 Tauri 应用窗口！

## 📋 这个模式的特点

### ✅ 优点
- 🖼️ 原生窗口体验
- ⌨️ 更好的快捷键支持
- 📁 更好的文件访问
- 🎨 更像真正的桌面应用

### ⚠️ 限制
- 🎬 视频还是 HTML5 播放
- 🚫 浏览器格式限制（H.265/EAC3 可能不支持）
- 🎯 但这是我们集成 libmpv 的第一步！
