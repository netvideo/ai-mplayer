# Privacy Policy

**Last Updated:** 2024

## 1. Introduction

This Privacy Policy explains how ai-mplayer ("we", "our", "the application") collects, uses, stores, and protects your personal information when you use our media player application.

## 2. Information We Collect

### 2.1 Local Data (Not Transmitted)

The following data is stored **locally on your device** and is **never transmitted** to external servers:

- **User Preferences**: Language settings, subtitle preferences, playback settings
- **Media Information**: File paths of recently played videos (local only)
- **Hardware Information**: CPU, GPU, and memory specifications (for optimization)

### 2.2 Optional AI Features (Local Processing)

When you enable AI features using **local Ollama**:

- **Audio Data**: Temporary audio segments for ASR (Automatic Speech Recognition)
- **Subtitle Text**: Text extracted from subtitle files for translation
- **Processing**: All AI processing happens locally on your machine
- **No Cloud Transmission**: No data is sent to external AI services by default

### 2.3 Optional Cloud AI (User Configured)

If you configure external AI endpoints:

- **API Keys**: Stored locally using encryption
- **Request Data**: Subtitle text or media metadata sent to configured endpoints
- **User Control**: You choose which external services to use

## 3. How We Use Information

### 3.1 Local Data Usage

- **Application Settings**: Remembering your preferences
- **Playback State**: Resuming videos where you left off
- **Performance Optimization**: Adjusting settings based on hardware

### 3.2 AI Features Usage

- **ASR**: Converting speech to text for subtitle generation
- **Translation**: Translating subtitle text to your preferred language
- **Video Enhancement**: Local video processing (if enabled)

## 4. Data Storage and Security

### 4.1 Local Storage

- **Location**: All data is stored in your local application data directory
- **Encryption**: Sensitive data (API keys) is encrypted using OS-level security
- **Access**: Only the application can access this data

### 4.2 No Remote Storage

- **No Cloud Sync**: We do not sync your data to cloud servers
- **No Analytics**: We do not collect usage analytics
- **No Telemetry**: We do not send diagnostic data

## 5. Your Rights (GDPR/CCPA Compliance)

### 5.1 Access Rights

You can access your stored data by:
- Opening the application settings
- Viewing the configuration files in the application data directory

### 5.2 Deletion Rights

You can delete your data by:
- Using the "Reset Settings" option in the application
- Manually deleting the application data directory
- Uninstalling the application

### 5.3 Data Portability

Your settings are stored in standard formats and can be:
- Exported as JSON files
- Backed up manually
- Transferred to another device

## 6. Third-Party Services

### 6.1 Optional External Services

You may choose to connect to:
- **Ollama**: Local AI inference server
- **Cloud AI APIs**: User-configured endpoints

### 6.2 Service-Specific Privacy

Please review the privacy policies of any external services you configure:
- [Ollama Privacy Policy](https://ollama.com/privacy)
- Review policies of any cloud AI providers you use

## 7. Children's Privacy

This application is not intended for children under 13. We do not knowingly collect personal information from children.

## 8. Security Measures
