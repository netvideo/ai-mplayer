## 技术方案

### 方案 A：使用纯 Rust 绑定（mpv crate）
- 使用 `mpv = "2.0"` crate
- 优点：类型安全，Rust 原生
- 缺点：可能有编译问题，需要正确配置

### 方案 B：使用 FFI 直接绑定（libmpv-sys）
- 使用 `libmpv-sys = "2.0"`
- 优点：更灵活，直接控制
- 缺点：需要手写 unsafe 代码

### 方案 C：混合方案（推荐）
- 使用 `mpv` crate 的高级 API
- 需要时使用 `libmpv-sys` 处理特殊情况
