## 🚀 测试步骤

### 方式 1: 纯前端模式（推荐先试这个）
```bash
npm install
npm run dev
```
然后在浏览器打开 http://localhost:1420

### 方式 2: Tauri 应用模式
```bash
npm install
npm run tauri:dev
```
